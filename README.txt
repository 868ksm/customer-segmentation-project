CUSTOMER SEGMENTATION PROJECT

This project uses Unsupervised Learning with KMeans clustering.

Technologies:
- Python
- Flask
- Pandas
- Scikit-learn
- KMeans Clustering

How to run:

1. Create venv:
python3 -m venv .venv

2. Activate:
source .venv/bin/activate

3. Install:
pip install -r requirements.txt

4. Train model:
python3 train_model.py

5. Run website:
python3 app.py
